# 🤖 AI Resume Polisher
A Generative AI tool built to help job seekers transform basic bullet points into high-impact, professional achievements using Llama 3.

## 🚀 Features
- Uses Groq API for lightning-fast LLM inference.
- Built with Streamlit for a clean, web-based UI.
- Context-aware rewriting based on target job titles.

## 🛠️ Tech Stack
- **Language:** Python
- **AI Model:** Llama 3 (via Groq)
- **Frontend:** Streamlit

## 📦 Installation
1. Clone the repo: `git clone [your-repo-link]`
2. Install requirements: `pip install -r requirements.txt`
3. Add your API Key to a `.env` file.
4. Run: `streamlit run app.py`
